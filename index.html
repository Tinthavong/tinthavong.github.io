Project Pathfinder

A different, gamified approach in visualizing search algorithms in Unity

The user can move the seeker and target objects and paint the board using brushes with weights that affect seeker's path to the target. The brown road is cheaper to traverse and the red wall is not traversable. The seeker can graze the wall due to a blur algorithm. This creates varying movement and “encourages” the seeker to favor the center of the road.

Causation

Gritty 2D action side scroller set in a cyberpunk world. A collaborative effort for Davenport University.

Developed in Unity, Causation tells the story of a family of bounty hunters whose actions affect the world. The player controls three characters, each with unique stories and gameplay. I handled version control with github and developed the core gameplay. I implemented character animations and created cutscenes with Unity's timeline sequencer.

Faceplant

Physics based 3D platforming game focused on parkouring and latching onto skyscrapers. A collaborative thesis project for Davenport University.

Developed in Unity, Faceplant is a first person 3D platformer. The player uses momentum from running, coupled with a grapple hook to navigate. I contributed by creating an NPC AI that has simple behavior for the player to latch onto and creating every 3D model. I utilized shaders in Unity to create animated clouds. I created and implemented a highscore database for the game.
